Hungry Hippos

Uh-oh..that purple hippo is starting to create a fuss! Let's use the grid-row-start property to feed her.

CSS grids use a two-dimensional layout, meaning it handles both columns and rows. The grid-row-start property functions like the grid-column-start property, except along the vertical axis. Why don't you try it out below?
